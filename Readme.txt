this is a simple and basic aircraft navigation system , controlled with keyboard.

it consists of 6 basic tools;

--artificial horizon (controlled withe the 4 arrows : left/right/up/down ) 
--cap conserver (controlled with: a/z )
--anemometer (controlled with: e/r )
--altimeter (controlled with: q/s )
--variometer (controlled with: d/f )
--turn coordinator (controlled with: c/v )


if you have no idea about any component of this system, you have to search and figure out.
